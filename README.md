# react-native-quick-websql

Fast WebSQL-compatible engine for React Native

## Installation

```sh
npm install react-native-quick-websql
```

## Usage

```js
import { multiply } from "react-native-quick-websql";

// ...

const result = await multiply(3, 7);
```
